# Prism Book Spectrum

> **Transform local folders into a visual bookshelf.**

Prism Book Spectrum is a local knowledge resource management application that transforms folders of books and documents into an interactive virtual bookshelf. Instead of browsing long file lists, users can organize, explore, and read their local resources through realistic book spines, categorized shelves, and an immersive reading interface.

## Features

* 📚 Virtual bookshelf with realistic book spines
* 🗂 Automatic folder-based classification
* 🎨 Prism-inspired color classification system
* 📖 Built-in reading support for PDF, EPUB, DOCX, TXT and Markdown
* 🌙 Light / Dark theme
* 📌 Reading progress and recent reading history
* 🔍 Search, sorting and category filtering
* 💻 Runs entirely on the local machine without uploading files

## Supported Formats

* PDF
* EPUB
* DOCX
* TXT
* Markdown (.md)
* MOBI (management)
* AZW3 (management)

## Getting Started

1. Download or clone this repository.
2. Open `index.html` in a modern browser.
3. Click **Select Folder**.
4. Choose your local book or document folder.
5. Browse your library through the virtual bookshelf.

## Project Vision

Prism Book Spectrum is more than an e-book reader.

Its goal is to transform scattered local documents into a structured, visual knowledge space, allowing users to observe, organize and explore their personal knowledge library more naturally.
